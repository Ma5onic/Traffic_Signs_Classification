
1- The images to train and test the model can be downloaded here:
http://www.nlpr.ia.ac.cn/pal/trafficdata/recognition.html

2- We worked with:
- TSRD-Train: 4170 images = 3127 train + 1043 test
- TSRD-Train Annotation: text file where each line = picture_name + the four coordinates of the panel + the category.

3- The images of TSDR-Train must be stored in the "data" folder, otherwise change the path in the code.

4- The annotations file can be stored in the parent directory of the notebook or change the path in the code.
